• Cisco Tools
    • Top Talkers shows the Netflow-reported top talker conversations from the routers. It updates every five minutes.
    • Interface Statistics allows you to choose whether you want to see Load statistics or Error statistics.
    • Run an ICMP Command allows you to ping or traceroute to a specific IP address.
    • Show Route allows you to query a router of your choosing to see where an IP / IPv6 address is routed.
    • Show ARP allows you to run an ARP query on an IP / IPv6 or MAC, and clear the entry as needed.
    • VLAN Trunking Change allows selected people to make trunking changes on Cisco routers.
    • Show Processes allows you to view the full process list, or the active processes only, on a selected router.
    • Show ACL shows you all ACLs on the selected router. You may click an ACL to see its contents.
    • Show Logs allows you to view the logs of a selected router.
  • F5 Tools
    • Show ARP allows you to run an ARP query on an IP or MAC, and clear the entry as needed.
  • VPN Tools
    • Session Stats shows you the information on three types of sessions. Click on a type to get more info about its active connections.
    • ITS Admin VPN Sessions allows you to check the session statistics for a specific mini-VPN.
  • Firewall Tools
    • Show ARP allows you to run an ARP query on an IP or MAC.
    • Run an ICMP Command allows you to ping or traceroute to a specific IP address.
    • Show Interfaces allows you to see the status of all interfaces on a firewall, or see statistics for a specific interface.
    • Show Route allows you to query the firewall for routing information for a specific IP address.
    • Show Log allows you to view the last 100 log entries for a JunOS firewall.
  • Tipping Point Tools
    • Show IP Blocks allows you to query the Tipping Point Core Controller to see if a specific IP is being blocked there.
    • Show DNS Blocks allows you to query the Tipping Points to see if a specific DNS name is being blocked there.
  • DHCP Tools
    • DHCP Scope Query allows you to check the DHCP scope configured for a given VLAN.
